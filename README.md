## Deskripsi Tugas
Tugas ini merupakan implementasi template untuk pengerjaan tugas Supervised Learning. Template ini mencakup langkah-langkah sistematis mulai dari import dataset hingga rangkuman hasil, dengan fokus pada analisis dan pemodelan data menggunakan algoritma machine learning.

## Tujuan Tugas
Tujuan dari tugas ini adalah untuk memberikan panduan lengkap bagi mahasiswa dalam menerapkan teknik Supervised Learning pada dataset nyata. Melalui template ini, mahasiswa diharapkan dapat memahami proses end-to-end dari eksplorasi data hingga evaluasi model, serta membandingkan performa berbagai algoritma machine learning.

## Model yang Digunakan
Tugas ini menggunakan tiga model Supervised Learning utama:
- **Decision Tree**: Algoritma pohon keputusan untuk klasifikasi atau regresi.
- **Naive Bayes**: Model probabilitas berbasis teorema Bayes, cocok untuk data dengan asumsi independensi fitur.
- **K-Nearest Neighbors (KNN)**: Algoritma berbasis jarak yang mengklasifikasikan berdasarkan kedekatan dengan tetangga terdekat.
